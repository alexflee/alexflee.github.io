Prototype kit techniques

A case study showing of my prototype kit techniques website, giving advice and techniques on how to create high-fidelity prototypes using the prototype kit.

Introduction

To help the rest of the design team at DVSA, I decided to put together some guidance on how I approach creating high-fidelity prototypes. The prototype contains example of how to use Javascript to simulate a realistic user journey, such as error messaging. It also includes an approach to A/B testing and creating unique journeys.

Visit the prototype website

The prototype is currently being hosted on heroku at prototype-techniques.herokuapp.com (link opens in a new tab). Use the password 'pr0t0typ3techniques' if you want to view the prototype!

I am planning to expand on these techniques as I continue to create prototypes!
